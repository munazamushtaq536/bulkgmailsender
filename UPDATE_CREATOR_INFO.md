## Quick Guide

### Step 1: Open the Dashboard File

1. Navigate to: `C:\Users\KCS\Desktop\Bulk Gmail Sender\public\dashboard.html`
2. Right-click → **Open with** → **Notepad** (or any text editor)

---

### Step 2: Find and Edit Creator 1 Info (Aqsa)

**Location:** Around **line 29-31**

**Find this:**
```html

<p class="creator-role">Full Stack Developer</p>

```

**Change to:**

## What You Can Change

### For Each Creator:

1. **Name** - Already set to "Aqsa" and "Munazza"
2. **Role** - Job title or position (e.g., "Software Engineer", "Product Designer", "Developer")
3. **Bio** - Short description about them (1-2 sentences)

## Quick Tips

- Keep roles **short** (2-4 words)
- Keep bios **brief** (1-2 sentences max)
- Be professional but friendly
- Save the file after making changes
- Always hard refresh (`Ctrl + Shift + R`) to see changes
